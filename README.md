# nodejs-express-react-mongo

A demo of using React + Node.js + MongoDB + Express + REST API

## Runing

Creat data folder
```bash
$ mkdir data
```

Start MongoDB server
```bash
$ mongod --dbpath data
```

Install dependencies
```bash
$ npm install
```

Start server
```bash
$ nodemon app.js
```

Open
```bash
$ open localhost
```
OR
```browser
http://localhost
```
